========================
欢迎使用杰理开源项目文档
========================

本开源项目文档系统是由杰理官方推出的针对杰理体系编译工具、软件结构、接口文档、生产测试工具的文档.

若需要查询更改和添加的信息发布,请咨询 :ref:`sdk_release_notes` 文档。

SDK 在 `Apache 2.0 许可`_ 下提供(在许可证文件中找到项目的 `GitHub 仓库`_ )。SDK还加载或重用包,脚本,使用其他许可和其他文件,描述在 :ref:`jl_licensing`

.. _Apache 2.0 许可:
   https://github.com/zephyrproject-rtos/zephyr/blob/main/LICENSE

.. _GitHub 仓库: https://github.com/Jieli-Tech/fw-AC63_BT_SDK
